# 光影瞬间 - 摄影作品集网站

一个精美的美的摄影作品集展示网站，采用现代UI设计，具有响应式布局和流畅的交互体验。

## 项目概述

"光影瞬间"是一个专注于展示摄影作品的单页网站，通过优雅的视觉设计和直观的用户界面，让访问者能够沉浸式欣赏摄影作品及其详细信息。网站支持作品分类筛选、优选作品展示、混合集浏览等功能，适配各种设备屏幕尺寸。

## 功能特点

- 响应式设计，完美适配移动端和桌面端
- 瀑布流布局展示摄影作品
- 作品分类筛选（人物、风景、自然、微距、建筑、静物等）
- 优选作品标记与筛选
- 作品详情模态框，展示拍摄参数等专业信息
- 混合集作品浏览功能，支持缩略图快速切换
- 平滑滚动和过渡动画，提升用户体验
- 自定义favicon和品牌色调

## 技术栈

- HTML5
- Tailwind CSS v3
- JavaScript (原生)
- Font Awesome 图标库
- 响应式设计与媒体查询

## 项目结构

```
PictureShow/
├── index.html           # 主页面
├── photos.json          # 作品数据
└── photos/              # 作品图片文件夹
    ├── [各类作品图片]
    └── sky/             # 天空系列混合集图片
```

## 作品数据格式

每个摄影作品的数据格式如下：

```json
{
  "id": 1,
  "title": "作品标题",
  "description": "作品描述文字",
  "category": "作品分类",
  "imageUrl": "图片路径",
  "isFeatured": true/false,  // 是否为优选作品
  "takenDate": "拍摄日期",
  "location": "拍摄地点",
  "camera": "相机型号",
  "lens": "镜头型号",
  "aperture": "光圈值",
  "shutterSpeed": "快门速度",
  "iso": "ISO值",
  "isCollection": true/false,  // 是否为混合集
  "collectionImages": ["图片1路径", "图片2路径"],  // 混合集包含的图片
  "collectionDescription": "混合集描述"  // 混合集描述文字
}
```

## 使用说明

1. 将项目文件部署到web服务器或本地环境
2. 在`photos`文件夹中添加摄影作品图片
3. 编辑`photos.json`文件，添加作品信息
4. 访问`index.html`即可浏览摄影作品集

## 交互说明

- 点击作品卡片查看详情
- 使用右下角筛选按钮进行作品筛选
- 在作品详情页，可通过左右箭头按钮或键盘方向键浏览混合集作品
- 点击关闭按钮或按ESC键关闭详情页
- 移动端可通过顶部菜单按钮切换导航菜单

## 设计亮点

- 采用芒果半熟渐变色调（#ffd700 到 #ff9f40）作为主题色，营造温暖氛围
- 模态框布局根据图片横竖比例自动调整，优化展示效果
- 图片加载和过渡动画提升用户体验
- 圆弧外框包裹拍摄参数，突出专业感
- 混合集缩略图设计，方便快速切换浏览

这个摄影作品集网站不仅是展示作品的平台，更是摄影师个人风格和审美理念的体现，通过精心设计的交互和视觉效果，让每一张照片都能传递出独特的"光影瞬间"。
